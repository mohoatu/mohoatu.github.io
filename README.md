# My Personal Website

Welcome to my personal website repository! This site is built with Jekyll and hosted on GitHub Pages.

## Features

- Clean, responsive design
- Built with Jekyll for easy customization
- Minima theme integration
- SEO optimized
- Mobile-friendly

## Quick Start

1. **Clone this repository** (if you forked it)
2. **Edit the files** to personalize your content
3. **Push to GitHub** - your site will automatically be published

## Customization

### Update Personal Information

Edit `_config.yml` to change:
- Site title and description
- Author name
- Social media links

### Modify Content

- **`index.html`**: Main landing page
- **`assets/css/style.css`**: Custom styling
- **`_config.yml`**: Jekyll configuration and theme settings

### Add Projects

Update the "Projects" section in `index.html` with your actual project details.

### Update Social Links

Edit the contact section and `_config.yml` with your actual social media URLs.

## View Your Site

Your site will be live at: `https://mohoatu.github.io`

It may take a few minutes for changes to appear after you push to GitHub.

## Theme

This site uses the **Minima** theme (GitHub Pages default) with custom CSS styling.
